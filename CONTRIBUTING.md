# Contributing

Improvements to the SKILL.md prompt are welcome — open a PR with before/after examples showing the change.

## How

1. Fork repo
2. Edit `skills/neandercode/SKILL.md` — this is the only copy you need to touch
3. Open PR with:
   - **Before:** what neandercode say now
   - **After:** what neandercode say with change
   - One sentence why change better

> **Note:** `neandercode/SKILL.md`, `plugins/neandercode/skills/neandercode/SKILL.md`, `.cursor/skills/neandercode/SKILL.md`, and `neandercode.skill` are auto-synced by CI after merge. Do not edit them directly.
> 
> **Note on compress skill:** If you are modifying the compress skill, edit `neandercode-compress/SKILL.md` or `neandercode-compress/scripts/`. CI will automatically sync these changes to `skills/compress/` and `plugins/neandercode/skills/compress/`.

Small focused change > big rewrite. Neandercode like simple.
