# Neandercode Compress

Run neandercode memory-file compression for a file path.

Process:
- If no path is provided in this message, ask for one absolute path and wait.
- Use the repo's `neandercode-compress` scripts.
- Execute: `python3 -m scripts <absolute_filepath>` from `neandercode-compress`.
- Summarize whether compression succeeded, was skipped, or failed.
